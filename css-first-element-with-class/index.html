然而，first 文字外的红色边框并没有出现。这是为什么呢？因为 :first-child 选择器的含义是【父元素的第一个子元素】，也就是说，上面的 CSS 会选择满足下列三个条件的元素：

  1. 祖先元素拥有 home 类
  2. 自己拥有 red 类
  3. 是其父元素的第一个子元素

显然，first 元素虽然满足条件1和条件2，但是并不满足条件3，因此是选不到它的。如果把 blah 去掉或者把 first 移到 blah 的上面，就满足这个条件了。
