常用参数说明：

  • -f --log-file=<logfile> 指定日志文件
  • -p --config-file=<filename> 指定配置文件
  • --real-time-html 实时更新
  • -a --agent-list 开启 UserAgent 列表。开启后会降低解析速度。
  • -d --with-output-resolver 输出 HTML 或者 JSON 报告时开启 IP 解析。
  • -o --output=<json|csv|html> 将给定文件重定向到标准输出，通过后缀名决定输出格式
