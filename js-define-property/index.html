Object.defineProperty

为对象定义属性：

可以通过 Object.defineProperty 这个方法，直接在一个对象上定义一个新的属性，或者是修改已存在的属性。最终这个方法会返回该对象。

descriptor 属性的描述配置，包括

  • 值，默认undefined
  • 可写（writable）
  • 枚举（enumerable）
  • 配置（configurable）

属性描述方法：

  • get
  • set
