JavaScript 中的 this

需要知道的几点：

  • 在严格模式和非严格模式下，this 的行为会有变化。
  • 在浏览器和 node.js 环境下，this 的行为会有变化。
  • 它是函数运行时自动生成的，只能在函数内部使用。

下面就是 this 的四种使用场景：

  1. this 指向调用该函数的对象

    Javascript 里的 this 看的是究竟是谁调用该函数，而不是看该函数被定义在哪个对象内。

  1. this 指向全局对象（浏览器是 window 对象、node.js 是 global 对象）

    若是使用 node file.js 这样的方式运行，this 并不会指向全局对象，非严格模式下会输出 undefined，而在严格模式下则会报错。
