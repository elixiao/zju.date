那var obj = {}和var obj = Object.create(null)一样吗？如果不一样，那它们有什么区别？

答案是不一样的：Object.create() 方法会使用指定的原型对象及其属性去创建一个新的对象，以字面量方式创建的空对象 var obj = {} 就相当于 var obj = Object.create(Object.prototype)，而Object.create(null) 不会继承任何对象，所以也不会有其他属性，包括 toString 方法。下面的代码验证了这一点：
