firebase 搭建静态网站

firebase

  • Firebase Realtime Database

    可以存储 JSON 应用数据（例如游戏状态或聊天消息），并立即在所有已连接设备间同步更改。

  • Firebase Remote Config

    可以在不要求用户下载更新的情况下存储开发者指定的键值对来更改您的应用的行为和外观。

  • Firebase Hosting

    可以托管您的网站的 HTML、CSS 和 JavaScript，以及开发者提供的其他资产，例如图形、字体和图标。
