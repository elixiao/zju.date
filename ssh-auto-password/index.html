ssh自动输入密码

假设管理员密码为 “12345”，要执行的 sudo vi test.txt，那我们就写一个 test.sh 档，里面的内容是：echo 12345 | sudo -S vi test.txt，主要就是把密码利用 echo 打在前面，后样 sudo 加上 -S，然后中间用 pipe 串起来即可。这个 -S 的意思是从 stdin 读取密码。
