服务器返回killed

在服务器上运行命令时突然中断，返回一个消息：killed，莫名其妙，网上查了下发现是服务器内存不足导致的，例如你的服务器只有512M的RAM，但是webpack的uglify插件需要2G的内存空间，超出限制，就会报上述错误。

有时候 npm install 的时候也是，服务器只有500M的内存。而且该项目会用到一些底层的npm包，会编译一些代码。这就导致了内存不够用。可以查一下相应的系统log，在路径 /var/log之下，确认自己的问题。如/var/log/syslog 和 /var/log/messages等。
