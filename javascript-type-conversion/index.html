JavaScript类型转换

JavaScript中的数据类型：

  1. string：字符串，例如：”hello”等
  2. number：数字，例如-1，3.14等
  3. boolean：布尔，只有true或false
  4. null：空的，没有值
  5. undefined：未定义，未声明或声明过但未赋值
  6. object：对象类型

除了object外其他都是基本类型。

转换基本类型

用的多的有两种情况：

  1. 一个表示数字的字符串转换为数字

  2. 其他基本类型转换为boolean

值 boolean值
true/false true/false
undefined false
Number 0和NaN转换为false，其他为true
String ‘’转换为false，其他为true

二、对象转换为number

对象和对象进行比较就看是否引用同一个对象，不是一个对象就是false，是同一个对象就是看它们valueOf的值。

除了这种比较之外，还有什么情况下需要把对象转换为数值呢？

  1. 当需要一个数值作为函数参数的时候。如：Math.sin(obj), isNaN(obj), +obj，当传入对象的时候，会把对象转换为number。
  2. 当对象和基本类型进行比较的时候。如：obj==’hello’
  3. 显示调用Number(obj)的时候。

所以当有需要把对象转换为数值的时候是这样的逻辑：

  1. 如果 valueOf 方法存在并且返回基本类型，那么就返回它。

  3. 否则报错。

由于很多对象没有valueOf方法，所以在转换为数值的时候都是调用toString方法的。
