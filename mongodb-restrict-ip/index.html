mongodb 指定 IP 访问

MongoDB 可以限制只允许某一特定 IP 来访问，有两种方法：

  • 方法一：在启动时加参数：mongod --bind_ip 127.0.0.1,10.0.133.14

  • 方法二：在 /etc/mongodb.conf 文件中添加以下内容：bind_ip = 127.0.0.1,10.0.133.14
