JavaScript中的this

JavaScript中的this和其他语言中的this不太一样，总共分为4种情况。

在全局环境下使用this

JavaScript中有一个在浏览器中就是window对象，在Node.js里叫全局对象，因此在全局代码下this就指的是这个全局对象。

call和apply函数里面的this

总结：

  1. 默认情况下，this指代全局变量
  2. 当一个函数通过父对象来调用，this指代父对象本身
  3. 当使用new操作符创建对象，this指代新创建的对象
  4. call和apply函数里面的this指代call或apply函数的第一个参数
