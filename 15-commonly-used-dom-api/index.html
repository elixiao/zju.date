JavaScript常用DOM接口

  1. querySelector()

    返回第一个满足给定CSS选择器的元素，如果没有符合元素则返回空（null）。

    • selector：CSS选择器，例如： “#fooId”, “.fooClassName”, “.class1.class2”, “.class1, .class2”
    • ele：第一个满足条件的元素或null

  2. querySelectorAll()

    返回满足选择器的所有元素，如果没有任何元素匹配则返回空对象（[]）

    • selector：CSS选择器
    • eles：所有满足条件的元素列表或空对象

  5. createElement()

    根据标签名创建HTML元素，例如：’p’ 和 ‘div’，然后可以通过 AppendChild() 等方法添加到页面上。

    • tagName：HTML标签名.

  6. appendChild()

    添加一个元素到给定元素作为其最后一个子节点，被添加的元素可以是新创建的元素，也可以是已存在的元素，如果是已存在的元素，那么就会从原来的位置删除。

    • ele：父元素
    • childEle：子元素

  7. removeChild()

    删除指定子元素。

    • ele：父元素
    • childEle：子元素

  8. replaceChild()

    从父元素中替换原来的子元素。

    • ele：父元素.
    • newChildEle：新子元素.
    • oldChildEle：旧子元素.

  9. cloneNode()

    克隆元素。

    • dupeEle克隆出来的元素
    • ele：被克隆的元素.
    • deep 可选参数，是否深度克隆，如果true则克隆所有子元素，否则只克隆元素自己。

  10. insertBefore()

    insertBefore方法将新元素插入到某个参考子元素的前面。如果这个参考子元素不存在，就把新元素作为最后一个子元素插入，效果和appendChild()一样。

    • ele：父元素
    • newEle：被插入的新元素
    • refEle：参考元素

  11. createDocumentFragment()

    createDocumentFragment()方法可用于一次性批量插入很多元素，它会创建一个DocumentFragment对象，这个对象是虚拟的DOM节点，并不在DOM树上，由于每次插入DOM节点都会触发一次重绘或重排，如果多次插入会降低性能，通过这个对象批量插入就可以只触发一次重绘或重排。

  12. getComputedStyle()

    用于获取指定HTML元素上的样式，该方法返回值是只读的。

    • style：CSSStyleDeclaration 对象，包含指定元素的所有CSS属性。
    • ele：指定元素
    • cpseudoEle：表示为元素，例如:after，可选。

  13. setAttribute()

    用于为指定元素创建一个新属性或覆盖旧属性的值。

    • ele：指定元素
    • name：属性名
    • value：属性值

  14. getAttribute()

    获取指定元素的某个属性值。

    • ele：指定元素
    • name ：属性名

  15. removeAttribute()

    删除指定元素的某个属性。

    • ele：指定元素
    • name：属性名
