max和min

max表示保留原始比例的前提下拉伸或缩放图片，且不超过指定尺寸。相反，min表示保留原始比例的前提下拉伸或缩放图片，且不少于指定尺寸。由于resize指定的尺寸比例不一定与图片原始比例相同，max完全嵌入resize尺寸的最大尺寸，min就是完全覆盖resize尺寸的最小尺寸。例如把一张图片拉伸或缩放图片，保持原始比例且min(width,height)=1024：
