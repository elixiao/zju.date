Linux hash命令

hash命令：

hash 命令的作用是在环境变量 PATH 中搜索命令 name 的完整路径并记住它，这样以后再次执行相同的命令时，就不必搜索其完整路径了，而且 shell 每次执行环境变量 PATH 中的一个命令时，hash都会记住它。

  • 不指定任何参数时，显示当前hash列表
  • -l 用于显示当前 hash 列表中的命令及完整路径等信息
  • -r 用于清空hash列表。
  • -p filename` 用于指定命令name的路径，路径 filename 是一个文件而非目录
  • -d 用于从hash列表中移除命令name对应的记录。
  • -t 用于显示hash列表中命令name的完整路径
