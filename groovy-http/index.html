groovy http

groovy script 程序中，撰写 http 网络程序例如网络爬虫、网页内容分析器等比较容易。介绍 groovy 的 URL 类别，http builder dsl 语法。

builder 是 groovy 程序常见的一种 DSL 支援。httpBuilder 为了支援 http 传输协议而设计的领域描述语言，方便进行 http request 和 response 的处理。HTTPBuilder 常见的 ContentType 有以下几种类型：

  • ANY
  • BINARY
  • HTML
  • JSON
  • TEXT
  • URLENC
  • XML

不指定的话默认为ANY类型，即自动检测服务器的 content-type，某些情况下需要让 http client 程序伪装成某种浏览器，修改request header 的 user-agent 属性，
