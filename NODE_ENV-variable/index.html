NODE_ENV环境变量

通过NODE_ENV可以来设置环境变量（默认值为development），一般我们通过检查这个值来分别对开发环境和生产环境下做不同的处理，可以在命令行中通过下面的方式设置这个值：

  • linux & mac: export NODE_ENV=production
  • windows: set NODE_ENV=production
