git 撤销

Git 有 4 个区：

  • 工作区（Working Area）
  • 暂存区（Stage）
  • 本地仓库（Local Repository）
  • 远程仓库（Remote Repository）

以及 5 种状态：

  • 未修改（Origin）
  • 已修改（Modified）
  • 已暂存（Staged）
  • 已提交（Committed）
  • 已推送（Pushed）
