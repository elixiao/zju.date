webpack打包react项目中遇到的一些问题

React找不到挂载点

原因是bundle.js在head里面引用，但react挂载的根元素在body里面，这个时候body里面的元素尚未渲染，就报错了。解决办法：把对bundle.js的引用放到body的最后即可。

参考链接：
http://stackoverflow.com/questions/26566317/invariant-violation-registercomponent-target-container-is-not-a-dom-elem

找不到findDOMNode函数

没有找到findDOMNode这个函数，说明没有引入该依赖，

参考链接：
https://facebook.github.io/react/docs/top-level-api.html#reactdom.finddomnode
