Nginx 中的 location

语法：

修饰符有四种：

  1. = ：精确匹配并终止搜索
  2. ~：区分大小写的正则表达式匹配
  3. ~*：不区分大小写的正则表达式匹配
  4. ^~：字符串打头匹配（若是最佳匹配，则不进行正则表达式匹配）

优先级顺序：

  1. 精确匹配(=)
  2. 字符串打头匹配(^~)
  3. 正则匹配(~ 或 ~*)
  4. 通用匹配(/)

两种正则当中，区分大小写的优先级高，也就是不带 * 的优先级高
