CSS伪元素

伪元素(Pseudo Element)，顾名思义“不是网页里面真正的元素”，但看起来和真正元素一样，是它必须依附在一个真实的元素上。

在 W3C 的定义里总共有五个伪元素 ( 其他仍在测试阶段 )，分别是

  • ::before
  • ::after
  • ::first-line
  • ::first-letter
  • ::selection

::before和::after

::before 是在原本的元素“之前”加入内容，::after 则是在原本的元素“之后”加入内容，两者的表现形式都是 display:inline-block。

angular组件中的public和private修饰符

angular 组件中变量的 public 和 private 修饰符会影响到 AoT 编译：

在 AoT 模式下模板里面不允许访问组件类中的 private 属性，因为在 JiT 模式下是编译后的 ES5 代码，而在 JavaScript 中并没有公有属性和私有属性一说，但 AoT 模式下 typescript 会对变量的访问范围作出限制。

建议所有的属性都设置为公有属性，在 constructor 里面注入的 service，因为一般只会在 component 中使用，故设置为私有属性。
