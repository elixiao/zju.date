什么意思呢？因为 typeRoots 指定了包含 type 定义的文件夹，type 指定了需要包含定义的名称，如果指定了 typeRoots，那么只有 typeRoots 文件夹下面的包才会被收录，如果指定了 types 那么只有那些在 typeRoots 文件夹下面存在的指定的包才会被收录，也就是说可能typeRoots 文件夹里面有："node", "lodash", "express" 3个包，如果只指定 node 那么就忽略另外两个。而可怕的是 "types": [] 是禁止自动引入任何包，它竟然是 angular-cli 项目中默认的选项，会给初学者带来很多烦恼。
