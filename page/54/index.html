Java相关名词解释

JVM

对Java程序而言，只认识一种操作系统，这个系统叫JVM，.class文档就是JVM的可执行文件。JVM担任的职责之一就是将.class文档翻译为各个操作系统（windows、linux、mac os）能看得懂的机器码。

JRE

JRE全称是Java Runtime Evironment，就是Java的执行环境，包含Java SE API和JVM，.class文件能够在JRE中执行。

JDK

JDK不仅包含了JRE，还包含了javac、javadoc等工具，对于要开发Java程序的人必须安装JDK。

PATH

.java文件必须编译为.class文件才能在JVM中执行，Java的编译工具程序是javac。装好JDK之后，工具程序就放在JDK安装文件夹的bin文件夹当中。当输入一个指令而没有指定路径信息时，操作系统会依照PATH环境变量中设定的路径顺序，依次寻找各个路径下是否有这个指令，如果不在PATH环境变量中就无法找到从而编译失败。注意：当系统安装两个以上的JDK时，系统将根据Path路径中设定的顺序决定执行哪个JDK下的工具程序。

CLASSPATH

实体操作系统执行某个命令时，会根据PATH中的路径信息视图找到可执行文件，而对于JVM来说，它的可执行文件就是.class文件，CLASSPATH就是告诉JVM我的.class文件路径在哪里。当执行java hello命令的时候， VM会依照CLASSPATH的路径顺序搜索这个hello.class这个文件，JVM预设的CLASSPATH就是当前文件夹。如果使用Java开发了链接库，这些类文件会被封装为JAR（Java archive），.jar后缀的文档实际使用zip格式压缩，里边包含一堆的.class可执行文件，于是需要在CLASSPATH当中指定。

groovy学习笔记

groovy的几条命令

  1. groovyc

    命令行启动groovy编译器，用于将groovy源文件编译为二进制文件，类似于Java当中的javac。例如：
    groovyc MyClass.groovy会产生MyClass.class文件

  2. groovysh

    启动groovy shell的命令行应用程序，可直接定义类、运行groovy语句、表达式等。例如：
    println "hello"直接打印出hello

    当然也可以直接定义一个类，例如：

    class Foo {
    def bar() {
        println "baz"
    }
}

    然后直接使用这个类：

    foo = new Foo()
foo.bar()

  3. groovyConcole

    启动groovy控制台窗口，可以写脚本。用Ctrl+Enter或Ctrl+R作为运行指令的快捷键。

groovy学习笔记

Groovy是动态类型，即变量所引用值的类型是在其运行过程中确定的，而不是编译时确定的。动态类型会使程序变得更简洁。

在程序代码任何地方，可以非常自由地给变量赋新值，从而拥有不同的数据类型。这是groovy的优点，同时也是它的弱点，因为用户可以自由地将一个string值赋给integer变量，但代码不会意识到此问题，从而导致其他难以预测的后果。

有一点值得注意：groovy中两个整数相除得到浮点数，这与其他语言不一样。

集合类型

列表（list）

列表是用来存储数据项集合的数据结构。[]操作符就是List类里面定义的getAt方法，所以：a[3]就是a.getAt(3)

当然，也可以通过索引范围来操作列表，将产生一个新的列表对象，初始值从start索引位置开始，到原始列表的end索引位置结束。使用索引操作符同样可以为列表赋值，例如a[1]=3或者a[1]=[3,4]

通过<<运算符可以把一个新元素追加到列表的最后，例如：a<<15，使用+操作符可以连接两个列表，使用-操作符可以从列表中删除元素。

列表方法：reverse、remove、pop等

映射（map）

映射是一种引用对象的无序集合，映射中的所有元素都通过关键字访问，关键字可以是任意类型的。

一个map里面包含key和value键值对，如果key的元素是一个变量名，那么它将被解释成一个String值。

映射方法：put、keySet等

范围（range）

范围是表达特定序列值的一种简略方法。
常用的是数字、字母（其实是acill码）

range也有一系列的方法。

输出：print和println两个方法用于基本输出，如果想要格式化输出，则使用printf(String format,List values)，format是格式，values是想要输出的值。

正则表达式

groovy支持使用~”regex”来定义正则表达式。def regex = ~'chesse'就创建了一个正则表达式。

groovy操作符”=~”用于使左边的操作数与右边的正则表达式匹配。
^用于表示开始
$用于表示结尾
+表示出现1次或多次
*表示出现零次或多次
{}用来匹配指定次数的字符
.表示任意字符（称为通配符wildcard character）
