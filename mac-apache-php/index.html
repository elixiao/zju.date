Mac中使用Apache运行PHP

PHP文件需要在 Apache 下运行，Mac 上自带 Apache，默认是关闭的，需要按照下面的方式开启：

  • 打开 /etc/apache2/httpd.conf

  • 找到下面两行，把 # 去掉

  • 启动 sudo apachectl start

然后浏览器中输入http://localhost，如果出现It works!就表示成功了。

常用 apache 命令

  • 启动：sudo apachectl start
  • 重启：sudo apachectl restart
  • 停止：sudo apachectl stop
  • 查看：sudo apachectl -v
