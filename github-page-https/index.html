github page 添加 https 支持

一直用 hexo 写博客，然后发布到 Github Page，现在 GitHub 宣布与 Let’s Encrypt 合作，为 Github Pages 自定义域名提供官方的 HTTPS 支持。配置 https 的方法如下：

  1. 先把 Custom domain 清空（假设原来是 zju.date）

    如果不清空的话，下面的 checkbox 是无法点击的：

  2. 如果是 CNAME 记录，请跳过这一步。如果是 A 记录，去域名服务商那里，把原来的域名 A 记录指向的 IP 地址（192.30.252.153 和 192.30.252.154） ，改成下面几个新地址之一：

    • 185.199.108.153
    • 185.199.109.153
    • 185.199.110.153
    • 185.199.111.153

    保存修改后，等待 DNS 记录生效。

  3. 把 Custom domain 改成原来的域名（改回 zju.date）

    改回之后，checkbox 可以点击了，但是会出现下面的提示：

    意思是证书还没申请好，暂时还无法用 https 访问。不过没关系，过一会就可以了。
