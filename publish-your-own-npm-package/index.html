发布自己npm包

  1. 编写模块

    一个最简单的NPM包由主模块index.js和包描述文件package.json组成。

  2. 注册包仓库账号：npm adduser

  3. 上传包：在目录下执行npm publish
  4. 安装包：npm install package_name

如果发布的包有问题，可以在 24h 内撤回：

撤回后不能再发布相同版本的包了。
