feathersjs hook

feathersjs hook 对象里面究竟都有什么？

  1. hook.app

    只读属性，是 feathersjs 对象，可用于获取其他 service，例如：hook.app.service('name')

  1. hook.service

    只读属性，当前 hook 执行所在的 service

  2. hook.path

    只读属性，表示当前 service 的路径

  3. hook.method

    只读属性，是 find, get, create, update, patch, remove 中的一个

  4. hook.type

    只读属性，是 before, after, error 中的一个

  5. hook.params

    可写属性，包含客户端传过来的参数，例如 params.query

  6. hook.id

    可写属性， get, remove, update 和 patch 方法才有

  7. hook.data

    可写属性，create, update 和 patch 方法才有

  8. hook.error

    可写属性，只有当 hook.type = error 时才有

  9. hook.result

    可写属性，只有当 hook.type = after 时才有，但是可以在 before 或 error hooks 中设置它。
