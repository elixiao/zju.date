使用 supervisor 管理进程

supervisor 是 Python 写的进程管理工具，可以很方便的用来启动、重启、关闭多个进程。

  • 不可以在 Windows 上运行
  • 可以运行在 Python 2.4 或者更高的 Python版本，但是不可以在任何 Python 3 的版本上运行。

    • $CWD/etc/supervisord.conf
    • /etc/supervisord.conf
    • /etc/supervisor/supervisord.conf
    • ../etc/supervisord.conf
    • ../supervisord.conf

注意有可能是下面两种情况：

  1. 程序使用了 daemon 模式
  2. 没有权限（例如使用的 sudo 需要密码）

1 的解决方法就是取消 daemon 模式，使用前台进程；2 的解决办法有三种，第一种是使用 sudo supervisord，第二种方法是把密码作为参数echo mypassword | sudo -S command，第三种改变应用程序的可执行权限sudo chmod +s app。
