Alpine Linux的优势和问题

  1. 优势

    • Alpine Linux的Docker镜像特点是轻巧（大小只有5M）且有完整的包管理工具(APK)。
    • Docker 官方镜像可能会用 Alpine Linux 替 换Ubuntu。

  2. 问题

    • Alpine Linux 使用了 musl，可能和其他 Linux 发行版使用的 glibc 实现会有些不同。
    • Alpine 里面没有 bash，只有 sh
