VBA学习笔记

变量

定义变量：

Dim 变量名称 [AS 数据类型]

在没有指定数据类型的时候，默认都是Variant类型。VBA的数据类型有：

数据类型 关键字 占用字节
布尔型 Boolean 2
日期型 Date 8
字符串 String -
货币 Currency 8
单精度 Single 4
双精度 Double 8
小数型 Decimal 14
字节型 Byte 1
整型 Integer 2
长整型 Long 4

符号常量：

Const 常量名称 [AS 数据类型]=常量值表达式

程序控制语句

判断语句：

第一种

If <条件表达式> Then <执行命令> End If

第二种

If <条件表达式> Then <执行命令1>  Else <执行命令2> End If

第三种

If <条件表达式1> Then 
<执行命令1> 
ElseIf <条件表达式2> Then 
<执行命令2>
…
Else 
<都不满足时执行>
End If

第四种

Select Case <条件表达式>
    Case  检验值1
    <执行命令1>
    Case 检验值2
    <执行命令2>
    …
    Case Else
    <都不满足时执行>
End Select

循环语句：
第一种

For 循环变量=初始值 To 终止值 [Step 步长]
    <执行语句>
Next [循环变量]

第二种

Do [{While|Until} condition]
    <执行语句>
    [Exit Do]
Loop

或者

Do
    <执行语句>
    [Exit Do]
Loop [{While|Until} condition]

第三种

For Each <元素> In <集合>
    <执行语句>
[Exit for]
Next [元素]

数组

声明数组：

Dim 数组名称(上界) AS 数据类型

默认从下标0到定义的上界，或者可以用：

Dim 数组名称(下界 To 上界) AS 数据类型

多维数组也是类似的：

Dim 数组名称(第一维上界，第二维上界) AS 数据类型

和

Dim 数组名称(下界 To 上界，下界 To 上界) AS 数据类型

数组初始化：

可以用循环来为数组进行一一赋值，也可以使用Array函数：

数组名称 = Array(数据集合)

动态数组中可保存的元素个数不确定，它的使用方法如下，首先声明：

Dim 动态数组名()

在使用的时候，使用ReDim重新定义数组大小：

ReDim [Preserver] 数组名称(界限)[AS 数据类型]

UBound和LBound函数可以获取数组的上下界。

函数

过程：

[Private|Public|Friend] [Static] Sub <过程名称> [(参数列表)]
    [语句段]
    [Exit Sub]
End Sub

函数：

[Private|Public|Friend] [Static]  <函数名称> [(参数列表)] [AS 数据类型]
    [语句段]
    [Exit Sub]
End Function

可以看到，过程和函数比较类似，但函数可以有返回值，在函数体内通过对函数名赋值来指定返回值。

过程和函数都可以通过在VBA代码中用函数名调用，除此之外，过程可以通过宏方式调用，函数可以在单元格中直接调用。

VBA的参数传递分按值传递(ByVal)和按地址传递(ByRef)，例如：

Sub Main()
    Dim x AS Integer
    x=25
    Debug.Print "主过程中变量x="&x
    testValue x
    Debug.Print "主过程中变量x="&x
End Sub

Public Sub testValue(ByVal x AS Integer)
    x=x+10
    Debug.Print "子过程中变量x="&x
End Sub

注意：数组只能按地址传递。

单元格操作

在VBA中常用Range和Cells表示单元格，有几种选择方式：

第一种

Range("A5").Select
Range("A"&"5").Select

第二种

Cells(5,1).Select
Cells(5,"A").Select

第三种

[A5].Select

如果想要选择某个工作簿某个工作簿某个单元格，可使用：

Workbook("工作簿名称").Worksheets("工作簿名称").Range("单元格地址")

连续单元格选择方式：

Range("A1:F10").Select
Range("A1","F10").Select
Range(Cells(1,1),Cells(10,6)).Select
[A1:F10].Select

如果选中某行或某列可以用：

Range("5:10")
Rows("5:10")
Columns("B:E")

with语句

with语句可以在一个单一对象或一个用户自定义类型上执行一系列语句：

With object
    [语句命令]
End With
