Mac设置默认浏览器

Mac设置默认浏览器的方法：

  • OS X Yosemite 以上版本：

    1. 点击 Apple () 菜单
    2. 选择系统偏好设置（System Preferences）
    3. 选择通用（General）
    4. 从默认网页浏览器（Default web browser）下拉菜单中选择。

  • Mavericks 以前版本：

    1. 打开 Safari
    2. 单击菜单选择偏好设置（Preferences）
    3. 选择通用（General）
    4. 从默认网页浏览器（Default web browser）下拉菜单中选
